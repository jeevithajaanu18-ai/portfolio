## 📝 Customization Quick Guide

### Update Name & Title
**React**: Edit `src/pages/HomePage.jsx` line ~40
**HTML**: Edit `portfolio.html` line ~520

### Change Colors
**React**: Modify `:root` variables in `src/App.css`
**HTML**: Modify `:root` variables in `portfolio.html` `<style>` tag

### Update Profile Photo
**React**: Edit image URLs in component files
**HTML**: Edit image URLs in `portfolio.html`

### Add New Projects
**React**: Edit `src/pages/ProjectsPage.jsx`
**HTML**: Duplicate project card in `portfolio.html`

### Modify Contact Info
Search and replace:
- `john.parker@example.com`
- `+1 (555) 123-4567`
- `San Francisco, CA`
