## 🎯 QUICK CUSTOMIZATION

### Change Your Name
Find and replace: `John Parker` → `Your Name`
- React: Files in `src/pages/` and `src/components/`
- HTML: Search in `portfolio.html`

### Update Profile Photo
Replace URL: `https://images.unsplash.com/photo-1507003211169-0a1dd7228f2d`
- Use your own photo URL
- Or any Unsplash image URL

### Change Contact Info
Replace these 3 items:
- Email: `john.parker@example.com`
- Phone: `+1 (555) 123-4567`
- Location: `San Francisco, CA`

### Modify Projects
1. Update project titles, descriptions, images
2. Add your actual project links
3. Change technology tags
4. React version allows filtering by category

### Change Colors
Edit the CSS color variables:
```css
--primary-color: #0066ff;      /* Change this blue */
--secondary-color: #ff6b6b;    /* Change this red */
--accent-color: #00d4ff;       /* Change this cyan */
```
